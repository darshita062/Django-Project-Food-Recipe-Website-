This project is a Fullstack Project. With a attractive UI.
It is a Food Recipe Website.
Build with Python Django Framework.
In this website we can add new recipes that will be stored in the database and are fetched from the database and displayed to all so that people can try your recipes....we can even read or give our feedback that is also stored in the database and visible to all users.
The frontend part is built using HTML,CSS,Bootstrap.
The Backend part is managed by dbSqlite a database provided by Django.
(TechStack : HTML,CSS,Bootstrap,Django,Python)
